# Seedream Exhibition Prompt Builder

## 节点信息

- 节点 ID：`SeedreamExhibitionPromptBuilder`
- 显示名称：`Seedream Exhibition Prompt Builder`
- 分类：`Seedream/Prompt`
- 输出：`prompt: STRING`

## 用途

该节点用于替代展厅工作流中由 `GPT Image Style Prompt Preset`、多个 `LazySwitchKJ` 和 `Text Concatenate` 组成的提示词拼接段。它会根据主题风格、参考图用途、色彩、语气、材料补充描述和 5 个布尔条件，生成更适合 Seedream 5.0 Pro 的自然语言结构化提示词。

当前节点只面向展厅写实渲染效果图，不做 GPT Image 双模式，也不修改旧 `GPT Image Style Prompt Preset` 的输出逻辑。

## 输入

### 固定输入

- `style_id`：主题风格。前端下拉显示中文标签，数据来自 `web/styles.json`，例如 `航天科技`、`商务`、`党建`。
- `tone`：画面语气，可选 `标准`、`明亮`、`稳重`、`高级`、`活泼`。
- `primary_color`：品牌强调色 Color Picker。最终提示词输出颜色语义和标准化 Hex，例如 `明亮科技蓝（#567DF0）`。
- `secondary_color`：辅助色 Color Picker。最终提示词输出颜色语义和标准化 Hex，例如 `低饱和蓝灰色（#D0D5DD）`。
- `base_prompt`：关闭主题模板时逐字符输出的原始提示词，不参与模板拼接。
- `additional_details`：材料、工艺、灯光、顶地墙等补充描述。空字符串会被忽略；单独的 Hex 值会被过滤。

### 布尔输入

5 个布尔项均固定暴露为可连线输入：

- `use_theme_template`：是否使用 `web/styles.json` 中的主题标签作为风格入口。
- `use_space_reference`：是否把参考图作为展厅空间布局与建筑结构依据。
- `include_people_placeholder`：是否将参考图中的占位人物或示意人形替换为写实游客。仅在 `use_space_reference=true` 时生效。
- `use_element_reference`：是否把参考图中的物品或元素作为元素参考。
- `lock_edit_region`：是否要求元素保持原位置。仅在 `use_element_reference=true` 时生效。

当前版本不提供 `visible_text` 输入，不会自动把补充描述包装为画面可见文字。需要在画面中出现的标语、标题或展板文字，建议作为独立提示词节点处理，并只对真实可见文字使用双引号。

## 拼接逻辑

`use_theme_template=false` 时，节点逐字符返回 `base_prompt`，不会清理或追加任何内容。

`use_theme_template=true` 时，节点忽略 `base_prompt`，按空行组织最多五个非空段落：

1. 输出目标：根据参考开关选择“基于参考图改造”或“创建展厅效果图”。
2. 参考约束：按开关输出空间、机位、透视、人物、元素和位置要求。
3. 设计与材质：输出主题空间设计语言，并追加有效的 `additional_details`。
4. 配色：输出主辅色的自然语言语义、Hex 和空间使用角色。
5. 灯光与摄影：输出 tone 对应的光线、曝光、对比度和写实质量要求。

## 色彩策略

节点保留 Color Picker 输入，并在提示词中同时输出中文语义和 `#RRGGBB`：

- 接近白色：`冷白色`
- 接近黑色：`深黑色`
- 灰阶：`浅灰过渡色`、`中性灰`、`深灰边界色`
- 蓝色系：`科技蓝`、`明亮科技蓝`、`高饱和科技蓝`
- 青色系：`冷光青蓝`、`柔和青色`
- 红色系：`主题红`、`稳重红色`
- 黄色系：`金色点缀`、`柔和暖黄色`
- 绿色系：`活力绿`、`柔和绿色`
- 紫色系：`科技紫`、`深紫色`

主题提供默认色，但用户输入拥有最终优先级。品牌强调色用于信息展示面、屏幕内容、发光线条和导视系统；辅助色用于过渡面、边界、留白和灯光层次。Hex 用于缩小模型对色相的理解范围，不代表像素级色彩保证。

## 主题模板

主题数据来自 `web/styles.json`：

- `label`：前端下拉显示名称。
- `subject`：模板目标段落使用的主题展厅名称。
- `prompt`：模板设计与材质段落使用的颜色中立空间设计语言。
- `thumbnail`：前端缩略图选择器使用的图片文件名。

## 前端行为

`web/prompt_preset.js` 为 `SeedreamExhibitionPromptBuilder` 提供缩略图风格选择器。`style_id` 在前端显示中文标签，同时后端仍兼容英文 style key。用户主动切换主题时会载入主题默认色；打开已有 workflow 或刷新页面时不会覆盖已保存颜色。用户主动清空两个文本输入时，前端会保留空字符串。

## 示例输出

选择 `航天科技`、主色 `#567DF0`、辅色 `#D0D5DD`，并启用全部参考开关时，输出会类似：

```text
基于参考图，将现有空间改造为航天科技展厅，生成用于展厅设计方案呈现的写实室内建筑渲染效果图。

保持参考图中的墙体、顶面、地面、空间尺度、动线关系和核心建筑结构，保持原有相机机位、视角、透视关系和画面构图。将参考图中的占位人物或示意人形替换为比例准确、姿态自然、正在参观展览的真实游客。保留参考图中已有展陈物件的位置、比例和空间关系，使其与改造后的环境自然融合。

采用精密构造线、轨道图形、数据可视化界面、线性照明和局部金属构件，形成克制专业的航天科技空间语言。地面以高抛光水磨石为主，结合局部PVC地材，墙面采用乳胶漆，搭配不锈钢、铝材和灯带装饰，顶面采用流线型连续灯带系统。

整体以低饱和中性色和浅色材质为基底。明亮科技蓝（#567DF0）作为局部品牌强调色，用于信息展示面、屏幕内容、发光线条和导视系统；低饱和蓝灰色（#D0D5DD）用于过渡面、边界、留白和灯光层次。墙面通过中性留白、材质分割和明暗变化形成节奏。

采用自然柔和的光线，曝光准确，对比度适中，构图均衡稳定。采用室内建筑摄影视角，表现真实材质反射，保持空间层次清晰、尺度合理，呈现高质量写实效果图质感。
```
